# RWA Tokenization Mini System

A simple end-to-end Real-World Asset (RWA) tokenization flow.

## ✅ Features
- Asset onboarding  
- Asset verification (Verifier role)  
- ERC-721 minting  
- Redemption/Burn flow  

## ✅ Stack
Solidity • OpenZeppelin • AccessControl

## ✅ Tokenization Steps (4-Step Flow)
1. Asset Onboarding  
2. Verification  
3. Minting  
4. Redemption / Settlement
